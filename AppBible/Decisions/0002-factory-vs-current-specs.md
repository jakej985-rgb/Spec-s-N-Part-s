# Factory Vs Current Specs
---
## Decision
- Factory specs are stored separately from current specs. 
- Factory data is never overwritten.
- Current specs repersent mods made by owner.
---
## Reason
- Supports engine swaps.
- Supports transmission swaps.
- Supports rearend swaps.
- Supports modifications.
- Preserves VIN accuracy.
- Allows the app to compare factory and current configurations.
---
## Status
- Accepted
