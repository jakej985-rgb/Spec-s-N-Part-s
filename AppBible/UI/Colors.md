# Colors

---

## Color Palette

### Primary Colors
- Primary Blue: #0066FF (actions, links, highlights)
- Dark Blue: #004BB3 (deep actions, headers)
- Light Blue: #E0F0FF (backgrounds, inactive states)

### Semantic Colors
- Success Green: #22C55E (confirmations, completed states)
- Warning Orange: #F59E0B (alerts, pending states)
- Error Red: #EF4444 (errors, critical actions)
- Neutral Gray: #6B7280 (text, dividers, disabled states)

### Backgrounds
- Light Mode:
  - Surface: #FFFFFF
  - Secondary Surface: #F9FAFB
  - Overlay: #000000 + 50% opacity (modals)
- Dark Mode:
  - Surface: #1F2937
  - Secondary Surface: #111827
  - Overlay: #FFFFFF + 30% opacity

### Text
- Light Mode:
  - Primary: #111827 (80% opacity for high contrast)
  - Secondary: #6B7280 (60% opacity)
  - Disabled: #D1D5DB (40% opacity)
- Dark Mode:
  - Primary: #F9FAFB (90% opacity)
  - Secondary: #D1D5DB (70% opacity)
  - Disabled: #6B7280 (50% opacity)

---

## Usage Guidelines
- Primary Blue: CTA buttons, active states, links
- Success Green: "Save", "Complete", confirmations
- Warning Orange: "Unsaved changes", pending syncs
- Error Red: "Delete", validation errors
- Gray: Labels, disabled buttons, dividers, helper text

---

## Contrast Requirements
- Text on background: minimum 4.5:1 (WCAG AA standard)
- UI components: minimum 3:1
