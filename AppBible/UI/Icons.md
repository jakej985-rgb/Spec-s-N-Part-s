## Usage
- 24x24px for most UI elements
- 32x32px for tab bar icons
- 48x48px+ for large buttons and focal points
- Stroke weight: 1.5-2px for readability

---

## Accessibility
- Icons always paired with text labels (or aria-label for SVG)
- Color alone should not convey status; use icon + text
- High contrast: minimum 3:1 against background
