# Layout

---

## Layout System
- 8pt baseline grid (all spacing in multiples of 8: 8px, 16px, 24px, 32px, 40px, 48px, 56px)
- Max width: 480px (mobile), 1024px (tablet/desktop)
- Safe area margins: 16px (mobile), 24px (tablet)

---

## Spacing Guidelines
- Padding (interior spacing):
  - Compact: 8px
  - Default: 16px
  - Comfortable: 24px
  - Spacious: 32px
- Margin (external spacing):
  - Between sections: 24px
  - Between elements: 16px
  - Between text and input: 8px

---

## Screen Layouts

### Single Column (Mobile)
- Full-width content with 16px side padding
- Title/header at top with 16px top padding
- Bottom tab bar (56px height, safe area aware)
- Content area between header and tab bar (scrollable)

### Two Column (Tablet/Desktop)
- Left sidebar (250px) or drawer
- Main content area (flexible)
- Right sidebar or panel (250px, optional)

---

## Component Spacing
- Card: 16px padding, 8px spacing between fields
- Button: 48px height min, 16px horizontal padding, 8-12px vertical padding
- Input field: 40px height min, 12px padding
- List item: 16px vertical padding, 16px horizontal padding

---

## Safe Areas & Notches
- Account for device notches and home indicator
- Inset content 8px from safe area edges
- Tab bar extends to bottom safe area

---

## Responsive Breakpoints
- Mobile: 0-480px
- Tablet: 481-768px
- Desktop: 769px+

---

## Typography Hierarchy
- H1 (heading): 28-32px, 700 weight, 40px line height
- H2 (section): 20-24px, 600 weight, 32px line height
- Body (default): 14-16px, 400 weight, 24px line height
- Caption: 12px, 400 weight, 18px line height
