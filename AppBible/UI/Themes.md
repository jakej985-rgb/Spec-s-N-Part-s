# Themes

---

## Theme Modes

### Light Theme (Default)
- Background: White (#FFFFFF)
- Text: Near-black (#111827)
- Accents: Primary Blue (#0066FF)
- Surfaces: Light gray (#F9FAFB)
- Suitable for: daytime, outdoor use

### Dark Theme
- Background: Near-black (#111827)
- Text: White (#F9FAFB)
- Accents: Light Blue (#60A5FA)
- Surfaces: Dark gray (#1F2937)
- Suitable for: nighttime, eye comfort

### System Theme (Auto)
- Follows device system preference
- Respects user's time-based schedule on supported devices
- Falls back to Light if system theme unavailable

---

## Theme Switching
- Available in Settings > Appearance.
- Persisted in UserPreferences.
- Applied immediately without app restart.
- Synced across devices.

---

## Component Theming
- Buttons: use theme-aware colors automatically
- Cards: shadow and border adjust per theme
- Text: color adjusts for contrast
- Icons: inherit theme color

---

## Accessibility in Dark Mode
- Sufficient contrast (4.5:1 minimum) maintained in all color pairs
- No pure black or pure white; use shades for eye comfort
- Colored elements tested for color-blind users

---

## Theme Transitions
- Fade animation over 200ms when theme changes
- No jarring flashes

---

## Future Ideas
