# Typography

---

## Font Family
- Primary: Inter (sans-serif, modern, readable)
- Fallback: system font (Segoe UI on Windows, -apple-system on Apple, Roboto on Android)
- Monospace: Courier New (for codes, VINs, part numbers)

---

## Font Sizes & Weights

### Display (Headlines)
- Display Large: 32px / 700 / 40px line height
- Display Medium: 28px / 700 / 36px line height

### Heading
- Heading 1: 24px / 600 / 32px line height (screen titles)
- Heading 2: 20px / 600 / 28px line height (section headers)
- Heading 3: 16px / 600 / 24px line height (subsection headers)

### Body
- Body Large: 16px / 400 / 24px line height (main text, descriptions)
- Body Medium: 14px / 400 / 22px line height (labels, secondary text)
- Body Small: 12px / 400 / 18px line height (captions, hints, metadata)

### Label
- Label Large: 14px / 500 / 20px line height (button text, input labels)
- Label Medium: 12px / 500 / 18px line height (tags, chips)
- Label Small: 10px / 500 / 16px line height (badges, minor labels)

---

## Font Weights
- 400 (Regular): body text, descriptions
- 500 (Medium): labels, buttons, emphasis
- 600 (Semibold): headings, section titles
- 700 (Bold): display, major headings

---

## Letter Spacing
- Tight: -0.5px (display text, dense information)
- Normal: 0px (body, most text)
- Relaxed: +0.5px (labels, captions)

---

## Line Height
- Tight: 1.2x (headlines)
- Normal: 1.5x (body, labels)
- Relaxed: 1.75x (long-form text, descriptions)

---

## Text Truncation
- Single line: ellipsis (...) for overflow
- Multi-line: max 3 lines, then ellipsis
- Example: vehicle name truncates at 20 characters or 1 line

---

## Accessibility
- Minimum 14px for body text (readability)
- Sufficient contrast (4.5:1) for all text on backgrounds
- Line height >= 1.5x for readability
- Letter spacing >= 0.12x for dyslexia-friendly reading

## Future Ideas
